# 9. AI辅读
人工智能助手，对当前文献进行智能摘要、段落解读、术语解释、研究脉络梳理，并支持问答互动——用户可直接提问，AI实时给出基于原文的答案与出处，大幅提升阅读效率与理解深度。使用AI辅读会消耗积分
- 支持一键总结全文大意；
