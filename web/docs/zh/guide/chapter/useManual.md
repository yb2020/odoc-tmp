# 1.使用手册（注册/登录）
## 1.1 注册
输入网址（https://odoc.ai）进入官网后，注册成功后方可使用。
![alt text](../../../image/1.1.png)
## 1.2 登录 
odoc.ai成功注册后，登录状态30日失效。若换电脑/浏览器或退出登录后，再次使用网页版的时候，重新登录邮箱和密码即可登录。
![alt text](../../../image/1.2.png)
![alt text](../../../image/1.2.png)
## 1.3忘记密码
若登录时忘记密码，可点击此处找回或重置密码。
