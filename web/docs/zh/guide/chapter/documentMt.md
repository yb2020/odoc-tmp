# 3. 文献管理
## 3.1 导入文献
目前只支持PDF格式的文献导入，不支持其他格式。

• 注：普通版上传文献一次最多只能30页，且普通版可上传每个文件最大限制为10MB，最大容量为20MB。专业版可上传文献最多100页，每个文件最大限制为100MB，最大容量为30960MB。

若不选择具体的文件夹，文献将被上传至【我的文献】里面。
有以下2种文献导入方式：

在导入文献前，点击【我的文献】选择具体的文件夹，点击【开始添加】可以将文献上传至该文件夹中；
![alt text](../../../image/3.1(1).png)
或者在【我的文献】处右键选择【添加文献】进行操作。
![alt text](../../../image/3.1(2).png)
![alt text](../../../image/3.1(3).png)
• 单篇文献导入：点击【添加】按钮，即可选择本地单篇文献导入到阅读器；
![alt text](../../../image/3.1(4).png)
• 多篇文献导入：点击【添加】按钮，按住shift键的同时即可选择本地多篇文献进行导入。
![alt text](../../../image/3.1(5).png)
## 3.2 新建文件夹
在【我的文献】里新建文件夹可以对文献进行分类管理：
• 在【全部文献】处右键点击【新建文件夹】按钮，输入文件夹名称，并点击确定。
![alt text](../../../image/3.2(1).png)
![alt text](../../../image/3.2(2).png)
• 新建子文件夹：在【我的文献】中点击打开已有文件夹，右键点击【新建子文件夹】（如果不选择已有文件夹，则新建文件夹默认建立为【我的文献】下方的一级文件夹）；
![alt text](../../../image/3.2(3).png)
• 文件夹导入文献：右键单击目标文件夹后点击【添加文献】或右键单击文献后选择【移动】到目标文件夹。（或点击文献后直接拖动到文件夹也能实现文件夹添加文献）
![alt text](../../../image/3.2(4).png)
![alt text](../../../image/3.2(5).png)
## 3.3 文件夹管理
鼠标移动到一个文件夹上单击【右键】可以实现文献管理功能，比如【新增】、【重命名】、【移动】、【复制】、【分享文件夹】、【删除】功能。
![alt text](../../../image/3.3.png)
## 3.4 文献题录信息查看
导入文献后，可自动呈现文献的题录信息：【文献名称】、【标签】、【作者】、【备注】、【发布日期】、【收录情况】、【JCR分区】、【影响因子】、【重要性】。
![alt text](../../../image/3.4.png)
## 3.5 题录信息筛选
点击右上角【多选类型】按钮，可以选择对文献列表的题录信息进行展示/不展示。
![alt text](../../../image/3.5.png)
## 3.6 题录信息修改
点击文献的【标签】、【作者】、【影响因子】、【JCR分区】、【发布时间】、【收录情况】、【重要性】可对题录信息进行修改。
![alt text](../../../image/3.6.png)
## 3.7 文献排序
暂未分类文献、自建文件夹中的文献可以通过点击【文献标题】、【影响因子】、【发布时间】、【重要性】右侧的排序按钮，选择文献按照文献标题首字母、影响因子数值高低、发布时间顺序、文献重要性指数的顺序/倒序进行排序：第一次点击后文献按照降序排列展示，第二次点击后文献按照升序排列展示，第三次点击后取消文献排序。
![alt text](../../../image/3.7.png)
## 3.8 文献搜索
可以在题目信息上方搜索栏通过输入【标题】，【标签】，【发布日期】，【备注】来快速检索文献，如果是搜索以作者为区分时，单击【作者】即可搜索。
![alt text](../../../image/3.8.png)
## 3.9 标签添加与筛选
### 3.9.1 添加标签
点击文献标题下方的【添加文献】按钮，可以选择为文献添加上系统标签/已有的自检标签/创建新的标签。一篇文献可添加多个标签。创建的新标签会保存在自建标签中。
![alt text](../../../image/3.9.1.png)
### 3.9.2 标签筛选
点击右上角【筛选标签】按钮，可以根据文献标签对文献进行筛选支持单选和多选。
![alt text](../../../image/3.9.2.png)
### 3.9.3 标签删除
在文献管理界面的标签处可以删除无用的文献标签，但目前不支持重新编辑现有的标签，可以删除旧标签后重新添加新标签。
![alt text](../../../image/3.9.3.png)
## 3.10 文献备注
在修改题录信息的页面，点击备注按钮，输入备注信息，备注信息将会被同步展示在文献标题右侧；
![alt text](../../../image/3.10.png)
## 3.11 文件夹排序
可手动拖拽文献列布页的文件夹进行自由排序或兼并为子文件夹。
![alt text](../../../image/3.11.png)
# 3.12文献阅读记录
在左侧工具栏中点击“最近阅读”即可查看文献阅读记录，根据阅读时间分为“今天”“昨天”“七日前”“本月更早”，也可在此界面上方添加文献。
