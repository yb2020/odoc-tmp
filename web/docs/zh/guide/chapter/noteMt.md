# 4.笔记管理
在【笔记列表】可以查看：【总结】、【单词本】、【摘录库】。

## 4.1 总结
阅读文献时所编写的总结在【我的笔记-总结】可以查看。

• 点击【我的笔记-总结-文献标题】即可跳转到该篇文献的总结文档进行查看和编辑；
![alt text](../../../image/4.1(1).png)
• 所有总结依据文献所在的文件夹分类，右边即可查看最近编辑时间。
![alt text](../../../image/4.1(2).png)
## 4.2 单词本
### 4.2.1单词本添加与查看
• 在文献中找到目标单词后，左键选中点击翻译，可在单词翻译界面选择添加到"单词本"
![alt text](../../../image/4.2.1(1).png)
![alt text](../../../image/4.2.1(2).png)
•单词翻译上方可自行选择不同翻译渠道，选择最合适的加入单词本。
![alt text](../../../image/4.2.1(3).png)
•在【我的笔记-单词/短语】可以查看所有收藏的单词、词性、释义、音标与发音；
![alt text](../../../image/4.2.1(4).png)
• 点击单词页右上方的【背诵模式】即可遮蔽该单词例句的中英文，帮助背诵；
![alt text](../../../image/4.2.1(5).png)
• 点击单词页左边的【来自文献】，可以查看该单词的来源文献。
### 4.2.2 释义修改与删除
• 点击需要编辑的单词的释义部分即可直接【编辑】，即可手动输入并修改单词释义；
![alt text](../../../image/4.2.2(1).png)
• 点击想要删除的单词右上角的按钮【删除】，即可删除该单词。
![alt text](../../../image/4.2.2(2).png)
## 4.3 摘录库
### 4.3.1 摘录定义
摘录在odoc.ai中定义为有写作参考价值的文献语句或段落。
收藏摘录即为在文献中收藏的文献语句或段落，自建摘录是对收藏摘录进行改写的文献语句或段落，以进行写作功底积累。
### 4.3.2 摘录库操作
点击【摘录库】可以查看收藏的摘录，可以对摘录做标签，也可以根据标签对摘录进行筛选。摘录也能做笔记、英译中、删除、标注阅读进度。
•【添加与查看摘录】：左键选中想摘录的语段，点击翻译后可添加至【摘录】;
![alt text](../../../image/4.3.2(1).png)
![alt text](../../../image/4.3.2(2).png)
或者选中摘录片段后点击编辑并添加标签后即可保存至摘录。
![alt text](../../../image/4.3.2(3).png)
![alt text](../../../image/4.3.2(4).png)
除文字编辑外，支持Markdown语法，支持ctrl/cmd+c/v粘贴图片。
![alt text](../../../image/4.3.2(5).png)
•【查看摘录】：点击文献阅读界面右边的【摘录】/【笔记-摘录】可以查看收藏的摘录。以下是两种查看方式：
![alt text](../../../image/4.3.2(6).png)
![alt text](../../../image/4.3.2(7).png)
•【摘录标签】：对于所有添加的摘录，都可以添加标签，输入内容后回车确认。或是直接选择之前的标签名称进行标记，
![alt text](../../../image/4.3.2(8).png)
![alt text](../../../image/4.3.2(9).png)
•【摘录内容查找】：所有标签将在页面上方展示。做标签以方便标记与查找。可以填写想要查找的摘录的部分内容关键词和标签到搜索框，点击搜索即可筛选出符合条件的对应的摘录。
![alt text](../../../image/4.3.2(10).png)
![alt text](../../../image/4.3.2(11).png)
•【摘录展示】：点击顶部的【展示规则】可以根据阅读进度选择；
![alt text](../../../image/4.3.2(12).png)
另外点击上方"按最新摘录"里的选项，将根据不同文献或同篇文献中的摘录收集时间来展示摘录；
![alt text](../../../image/4.3.2(13).png)
•【来源文献】：点击摘录下方的文献pdf，即可查看该摘录的来源文献。
![alt text](../../../image/4.3.2(14).png)
![alt text](../../../image/4.3.2(15).png)
•【摘录删除】：点击摘录右上方的【删除】按钮，即可删除该条摘录。
![alt text](../../../image/4.3.2(16).png)
•【摘录标签修改】：点击在【本篇摘录】或【全部摘录】中点击已添加的摘录，即可一键在改写框中可以编辑摘录内容，点击确定后会保存在【本篇摘录】和【笔记-摘录】中。除文字编辑外，支持Markdown语法，支持ctrl/cmd+c/v粘贴图片。
![alt text](../../../image/4.3.2(17).png)

## 4.4术语库
### 4.4.1 术语添加方式
选中文献中想要添加的术语单词，可选择"术语库"，并手动添加原词与自定义译文，可自定义翻译结果。
![alt text](../../../image/4.4.1(1).png)
![alt text](../../../image/4.4.1(2).png)
### 4.4.2 术语编辑或删除
选中任意单词单击翻译进入术语库后可选择单词，点击"编辑"即可改动译文，点击删除即可删除该术语。
![alt text](../../../image/4.4.2.png)
### 4.4.3 术语库的同步和查看
选中任意单词单击翻译进入划词翻译界面后开启术语库，即可同步术语词译文，点击"术语库"即可查看该文献中添加的所有术语词。
![alt text](../../../image/4.4.3.png)
# 4.5笔记标签
## 4.5.1【添加标签】
文献阅读时，长按鼠标左键选中一个长句后松开左键，然后下方出现【编辑图案】的按钮，即可添加标签后保存为摘录。
![alt text](../../../image/4.5.1(1).png)
![alt text](../../../image/4.5.1(2).png)
## 4.5.2【删除标签】
对于所有添加标签的摘录，在文献阅读页面鼠标经过时均会显示，点击图标删除。或者在【笔记-摘录】页面上方的标签点击标签删除内容后按回车键即可对标签进行删除。
![alt text](../../../image/4.5.2(1).png)
![alt text](../../../image/4.5.2(2).png)
## 4.5.3【重命名】
对于所有添加标签的摘录，在【笔记-摘录】页面上方的标签右键都可以进行重命名。或者在在文献阅读页面鼠标经过该摘录时展示出标签后，只能"先删除标签再重新添加标签"的方式来重命名标签。
![alt text](../../../image/4.5.3(1).png)
